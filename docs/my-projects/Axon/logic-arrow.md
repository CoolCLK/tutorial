**逻辑箭头** *(Logic Arrow)* 是 **Axon 语言** 最基础、最关键的语句。

**逻辑箭头** 有以下特点:

- 简单
- 绝对
- 可含参
- 可拓展

以下是常见的 **逻辑箭头** :

| 关键字  | 作用                                  |
|------|-------------------------------------|
| `->` | 使当前方法跳转到目标方法，完成运行后跳转回原方法            |
| `=>` | 并行运行方法 *(若目标方法未运行完毕，则该语句将阻塞原方法)*    |
| `+>` | 交叉运行方法 *(两个方法互不影响，原方法在执行完该语句后继续运行)* |
