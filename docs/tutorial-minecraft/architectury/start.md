:::tip

在使用 **Architectury** 之前，
推荐您先学习 **Fabric**、**NeoForge**、**Quilt**、**Mixin** 等相关教程。

:::

**Architectury** 是一个旨在简化多平台模组开发的工具链，
一份代码可以在多平台上运行！

不妨先从 [生成模板](generate.md) 开始，
一步步了解贯穿在 Architectury 的接口思想，
让其成为您强大的开发利器！

:::note

**Architectury** 自 `1.20.4` 开始，开始支持 **NeoForge**。
同时，自 `1.20.5` 开始，其不再支持 **Forge**。

:::
