### 题目描述

有一只小鱼，它平日每天游泳 $250$ 公里，周末休息（实行双休日），假设从周 $x$ 开始算起，过了 $n$ 天以后，小鱼一共累计游泳了多少公里呢？

### 输入格式

输入两个正整数 $x,n$，表示从周 $x$ 算起，经过 $n$ 天。

### 输出格式

输出一个整数，表示小鱼累计游泳了多少公里。

### 输入输出样例 #1

### 输入 #1

```
3 10
```

### 输出 #1

```
2000
```

### 说明/提示

数据保证，$1\le x \le 7$，$1 \le n\le 10^6$。

### 题目解答

小鱼是从周 $x$ 开始的，不妨记这一周是第 $1$ 周。
之后过了 $n$ 天，相较于第 $1$ 周的周 $1$ 就过了 $x+n$ 天。
那这之间有多少天假呢？就很明了了，应该是 $\lceil\frac{x+n}{7}\rceil\times 2$天吧，那当然，如果 $x\geq 6$ 这里还需要减掉 $7-x$ 天；如果 $((x+n-1)\mod 7)+1\geq 6$ 这里还需要减掉 $6-((x+n-1)\mod 7)$ 天。

那我们来编写代码吧。

```cpp showLineNumbers
/** 
  * @author clk
  */

#include <cstdio>

typedef unsigned int uint;

int main() {
  uint x, n;
  scanf("%u %u", &x, &n);

  uint weekends = (n / 7) * 2;
  uint r = n % 7;

  if (r > 0) {
    if (x <= 5) {
      if (r > 6 - x) weekends++;
      if (r > 7 - x) weekends++;
    } else if (x == 6) {
      weekends++;
      if (r > 1) weekends++;
    } else {
      weekends++;
    }
  }

  uint workdays = n - weekends;
  printf("%u\n", workdays * 250);
  return 0;
}
```
