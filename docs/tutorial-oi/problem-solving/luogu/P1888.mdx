### 题目描述

输入一组勾股数 $a,b,c（a\neq b\neq c）$，用分数格式输出其较小锐角的正弦值。（要求约分。）

### 输入格式

一行，包含三个正整数，即勾股数 $a,b,c$（无大小顺序）。

### 输出格式

一行，包含一个分数，即较小锐角的正弦值。

### 输入输出样例 #1

### 输入 #1

```
3 5 4
```

### 输出 #1

```
3/5
```

### 说明/提示

数据保证：$a,b,c$ 为正整数且 $\in [1,10^9]$。

### 题目解答

这里引用一下 [OI Wiki](https://oi-wiki.org/) 对最大公约数的计算证明[过程](https://oi-wiki.org/math/number-theory/gcd/#%E8%BF%87%E7%A8%8B)：

<details>
<summary>证明</summary>

设 $a=bk+c$，显然有 $c=a \bmod b$．设 $d \mid a,~d \mid b$，则 $c=a-bk, \frac{c}{d}=\frac{a}{d}-\frac{b}{d}k$．

由右边的式子可知 $\frac{c}{d}$ 为整数，即 $d \mid c$，所以对于 $a,b$ 的公约数，它也会是 $b,a \bmod b$ 的公约数．

反过来也需要证明：

设 $d \mid b,~d\mid (a \bmod b)$，我们还是可以像之前一样得到以下式子 $\frac{a\bmod b}{d}=\frac{a}{d}-\frac{b}{d}k,~\frac{a\bmod b}{d}+\frac{b}{d}k=\frac{a}{d}$．

因为左边式子显然为整数，所以 $\frac{a}{d}$ 也为整数，即 $d \mid a$，所以 $b,a\bmod b$ 的公约数也是 $a,b$ 的公约数．

既然两式公约数都是相同的，那么最大公约数也会相同．

所以得到式子 $\gcd(a,b)=\gcd(b,a\bmod b)$

</details>

最后我们可以得到代码：

<Tabs>
<TabItem value="cpp" label="C++" default>

```cpp
int gcd(int a, int b) { return b == 0 ? a : gcd(b, a % b); }
```

</TabItem>

利用 `gcd` 方法，我们就可以轻松地得到代码：

<Tabs>
<TabItem value="cpp" label="C++" default>

```cpp showLineNumbers
/** 
  * @author clk
  */

#include <cstdio>
#include <algorithm>

typedef unsigned int uint;

int gcd(int a, int b) { return b == 0 ? a : gcd(b, a % b); }

int main() {
  uint a, b, c;
  scanf("%u %u %u", &a, &b, &c);
  uint d = std::min({a, b, c});
  uint e = std::max({a, b, c});
  uint f = gcd(d, e);
  d /= f;
  e /= f;
  printf("%u/%u\n", d, e);
  return 0;
}
```
