### 题目描述

输入一个小写字母，输出其对应的大写字母。例如输入 q[回车] 时，会输出 Q。

### 输入格式

无

### 输出格式

无

### 输入输出样例

输入 #1
```
q
```

输出 #1
```
Q
```

### 题目解答

<Tabs>
<TabItem value="cpp" label="C++" default>

```cpp showLineNumbers
/** 
  * @author clk
  */

#include <cstdio>

int main() {
  char c;
  scanf("%c", &c);
  printf("%c\n", char(c - 32));
  return 0;
}
```
