## 题目描述

小 B 喜欢吃苹果。她现在有 $m$（$1 \le m \le 100$）个苹果，吃完一个苹果需要花费 $t$（$0 \le t \le 100$）分钟，吃完一个后立刻开始吃下一个。现在时间过去了 $s$（$1 \le s \le 10000$）分钟，请问她还有几个完整的苹果？

## 输入格式

输入三个**非负整数**表示 $m, t, s$。

## 输出格式

输出一个整数表示答案。

## 输入输出样例 #1

### 输入 #1

```
50 10 200
```

### 输出 #1

```
30
```

## 说明/提示

如果你出现了 RE，不如检查一下被零除？

### 题目解答

在 $t=0$ 时，就直接得到答案 $0$ 个了*（秒吃也是神了）*。

注意审题，是还有几个**完整的苹果**，所以不完整的与被吃掉的苹果数为 $\lceil\frac{s}{t}\rceil$。

<Tabs>
<TabItem value="cpp" label="C++" default>

```cpp showLineNumbers
/** 
  * @author clk
  */

#include <cstdio>
#include <cmath>

typedef unsigned short ushort;

int main() {
  ushort m, t, s;
  scanf("%hu %hu %hu", &m, &t, &s);
  ushort a = ceil(static_cast<double>(s) / t);
  printf("%hu\n", t > 0 && m > a ? m - a : 0);
  return 0;
}
```
