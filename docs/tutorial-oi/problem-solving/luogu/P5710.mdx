### 题目描述

一些整数可能拥有以下的性质：

- 性质 1：是偶数；
- 性质 2：大于 $4$ 且不大于 $12$。

小 A 喜欢这两个性质同时成立的整数；Uim 喜欢这至少符合其中一种性质的整数；小 B 喜欢刚好有符合其中一个性质的整数；正妹喜欢不符合这两个性质的整数。现在给出一个整数 $x$，请问他们是否喜欢这个整数？

### 输入格式

输入一个整数 $x(0\le x \le 1000)$

### 输出格式

输出这 $4$ 个人是否喜欢这个数字，如果喜欢则输出 `1`，否则输出 `0`，用空格分隔。输出顺序为：小 A、Uim、小 B、正妹。

### 输入输出样例 #1

### 输入 #1

```
12
```

### 输出 #1

```
1 1 0 0
```

### 题目解答

在某些语言中（如 C、C++），布尔与整数型是同一类型的数值，
那么我们记性质1、2为事件$A$、$B$，
每个人喜欢状态就是：
- 小 A：`A && B` 或 `A + B == 2`
- Uim：`A || B` 或 `A + B >= 1`
- 小 B：`(A && !B) || (!A && B)` 或 `A + B == 1`
- 正妹：`!(A && B)`、`!A && !B` 或 `A + B == 0`

/**

#include <cstdio>

typedef unsigned short ushort;

int main() {
  ushort x;
  scanf("%hu", &x);
  bool a = x % 2 == 0, b = x > 4 && x <= 12;
  printf("%d %d %d %d\n", a && b, a || b, a + b == 1, !a && !b);
  return 0;
}
